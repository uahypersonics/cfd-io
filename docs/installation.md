# Installation

## Requirements

- Python 3.11 or later

## From PyPI

```bash
pip install cfd-io
```

For Tecplot binary (.plt) support:

```bash
pip install cfd-io[tecplot]
```

!!! note
    This requires a valid Tecplot license.

## From Source

```bash
git clone https://github.com/uahypersonics/cfd-io.git
cd cfd-io
pip install -e .
```

## Development Installation

```bash
pip install -e ".[dev]"
```
